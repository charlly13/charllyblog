# **App Name**: Cyber Cloudfolio

## Core Features:

- About Me Section: A detailed section about the IT student's academic focus at New Era University, relevant coursework, and career aspirations.
- Projects Showcase: Display of cloud and cybersecurity projects with descriptions, images, and links to GitHub or live demos.
- Skills Matrix: Visualization of the student's skills in cloud computing, cybersecurity, and other IT areas.
- Contact Form: An interactive contact form for visitors to send messages directly to the student.

## Style Guidelines:

- Background color: Dark charcoal (#121212) to provide a modern, sophisticated feel.
- Primary color: Baby pink (#F4C2C2) to add a gentle, modern accent color for headings and interactive elements.
- Accent color: Pale rose (#FFE9E9) for highlights and subtle details to soften the contrast in the dark theme.
- Body text: 'Inter', a sans-serif font providing a modern and neutral feel for readability.
- Headline font: 'Space Grotesk', a sans-serif font that offers a techy, computerized feel for headings and titles.
- Use outline icons with a subtle baby pink (#F4C2C2) hover effect to maintain the dark theme aesthetic while providing interactive cues.
- Implement a clean, grid-based layout with ample whitespace to ensure readability and highlight key projects and skills.
- Subtle animations, such as fade-in effects on scroll and smooth transitions between sections, to enhance user experience without being distracting.
